## ✨ Features

* **Modern UI:** লাল এবং সাদার সমন্বয়ে বৈশাখী থিম।
* **Typing Animation:** ডাইনামিক ভাবে বাংলা শুভেচ্ছা বার্তা ভেসে উঠবে।
* **Confetti Blast:** বাটন ক্লিক করলে রঙিন কনফেটির মাধ্যমে উৎসবের আমেজ।
* **Fully Responsive:** মোবাইল, ট্যাবলেট এবং পিসি—সব ডিভাইসেই সুন্দর দেখাবে।
* **Lightweight:** কোনো ভারি ফ্রেমওয়ার্ক ছাড়াই তৈরি, তাই লোড হয় খুব দ্রুত।

---

## 🛠️ Technologies Used

* **HTML5:** প্রজেক্টের মূল কাঠামোর জন্য।
* **CSS3:** গ্লাস-মরফিজম কার্ড ডিজাইন এবং অ্যানিমেশনের জন্য।
* **JavaScript (ES6):** লজিক এবং ইন্টারঅ্যাকশনের জন্য।
* **Canvas-Confetti:** চমৎকার কনফেটি ইফেক্টের জন্য ব্যবহৃত লাইব্রেরি।
